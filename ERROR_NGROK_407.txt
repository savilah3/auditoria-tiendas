Error: "407 authentication required"
Causa: El proxy de Walmart (sysproxy.wal-mart.com:8080) requiere 
       autenticación NTLM/Kerberos que ngrok NO soporta bien.

¿Por qué funciona?
✅ De Microsoft → Soporta autenticación Windows integrada
✅ Funciona BIEN en proxies corporativos
✅ Gratis
✅ Integrado con tu cuenta Microsoft/Walmart

Instalación (1 minuto):

OPCIÓN A: Con winget (recomendado)
   winget install Microsoft.devtunnel

   1. Ve a: https://aka.ms/devtunnels/download
   2. Descarga "Windows CLI"
   3. Instala

   Doble click en: CREAR_TUNEL_DEVTUNNEL.bat
   
   → Te pedirá login con tu cuenta Microsoft
   → Obtendrás URL tipo: https://abc123.devtunnels.ms
