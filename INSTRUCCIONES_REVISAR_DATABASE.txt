🎯 OBJETIVO: Ver si tu app usa Supabase o Render PostgreSQL

⏱️ TIEMPO: 2 minutos

═══════════════════════════════════════════════════════════════════════

📱 PASO 1: Abrir Render Dashboard
═══════════════════════════════════════════════════════════════════════

1. Abre tu navegador
2. Ve a: https://dashboard.render.com
3. Login con tu cuenta (GitHub/email)
4. ✅ Deberías ver tu dashboard

═══════════════════════════════════════════════════════════════════════

🔍 PASO 2: Encontrar tu Web Service
═══════════════════════════════════════════════════════════════════════

En el dashboard verás algo como:

┌─────────────────────────────────────────────────────────┐
│  Web Services                                           │
├─────────────────────────────────────────────────────────┤
│  📦 auditoria-tiendas                                   │
│     https://auditoria-tiendas.onrender.com             │
│     Last deployed: X minutes ago                        │
└─────────────────────────────────────────────────────────┘

1. CLICK en "auditoria-tiendas" (o como se llame tu app)

═══════════════════════════════════════════════════════════════════════

⚙️ PASO 3: Ir a Environment Variables
═══════════════════════════════════════════════════════════════════════

En la página de tu web service verás tabs arriba:

┌──────────────────────────────────────────────────────┐
│  Events | Logs | Shell | Environment | Settings     │
└──────────────────────────────────────────────────────┘

1. CLICK en "Environment"

═══════════════════════════════════════════════════════════════════════

🔎 PASO 4: Buscar DATABASE_URL
═══════════════════════════════════════════════════════════════════════

Verás una lista de variables de entorno:

┌─────────────────────────────────────────────────────────────┐
│  Environment Variables                    [+ Add Variable]  │
├─────────────────────────────────────────────────────────────┤
│  Key                    Value                               │
├─────────────────────────────────────────────────────────────┤
│  PYTHON_VERSION         3.12.0                              │
│  DASHBOARD_USER         admin                               │
│  DASHBOARD_PASS         walmart2025                         │
│  DATABASE_URL           postgres://user:pass@XXXXX...       │
│                         👆 ESTA ES LA QUE NOS INTERESA      │
└─────────────────────────────────────────────────────────────┘

1. Busca la línea que dice "DATABASE_URL"
2. El valor estará oculto o visible parcialmente

═══════════════════════════════════════════════════════════════════════

👁️ PASO 5: Ver el valor completo (IMPORTANTE)
═══════════════════════════════════════════════════════════════════════

El valor puede estar oculto con "•••••••••"

1. CLICK en el ícono del "ojo" 👁️ al lado del valor
   O
2. CLICK en "Edit" o "Reveal" para mostrar el valor

Ahora deberías ver algo como:

postgres://usuario:password@XXXXXXXXX.supabase.com:5432/postgres

                               👆👆👆👆👆👆👆👆
                          ESTA PARTE ES LA CLAVE

═══════════════════════════════════════════════════════════════════════

📋 PASO 6: Copiar SOLO el host (la parte importante)
═══════════════════════════════════════════════════════════════════════

NO copies la contraseña completa, solo necesito el HOST:

Ejemplo de DATABASE_URL:
postgres://myuser:mypass@aws-0-us-west-1.pooler.supabase.com:5432/postgres
                          ^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
                          COPIA SOLO ESTA PARTE

Variantes posibles:

OPCIÓN A (Supabase):
  @aws-0-us-west-1.pooler.supabase.com
  @db.XXXXXXXXX.supabase.co
  
OPCIÓN B (Render):
  @dpg-XXXXXXXXX.oregon-postgres.render.com
  @dpg-XXXXXXXXX-a.oregon-postgres.render.com
