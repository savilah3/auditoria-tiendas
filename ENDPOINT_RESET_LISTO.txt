❌ SI EL ENDPOINT DA ERROR
═══════════════════════════════════════════════════════════════════

Posibles errores:

ERROR 1: "Connection refused" o "Database not found"
   → La BD de Render no está configurada
   → Necesitas verificar DATABASE_URL en variables de entorno

ERROR 2: "Permission denied"
   → El usuario de BD no tiene permisos para DROP/CREATE
   → Necesitamos hacerlo desde Render Console directamente

ERROR 3: Otro error
   → Cópiame el error exacto y lo arreglamos
