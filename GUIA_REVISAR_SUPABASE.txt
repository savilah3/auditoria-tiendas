🎯 OBJETIVO: Ver si tu app usa Supabase PostgreSQL

⏱️ TIEMPO: 1-2 minutos

═══════════════════════════════════════════════════════════════════════

📱 PASO 1: Abrir Supabase
═══════════════════════════════════════════════════════════════════════

1. Abre tu navegador
2. Ve a: https://supabase.com
3. Click en "Sign In" (arriba derecha)
4. Login con GitHub o email (el que usaste para crear cuenta)

═══════════════════════════════════════════════════════════════════════

🏠 PASO 2: Ver tus proyectos
═══════════════════════════════════════════════════════════════════════

Después de login verás algo como:

┌─────────────────────────────────────────────────────────┐
│  Supabase                                    [+ New]    │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  📦 Your Projects                                       │
│                                                         │
│  ┌──────────────────────────────────────────┐          │
│  │ 🗄️  auditoria-db                         │          │
│  │     Region: East US                      │          │
│  │     Created: X days ago                  │          │
│  └──────────────────────────────────────────┘          │
│                                                         │
│  o puede estar vacío si no tienes proyectos            │
│                                                         │
└─────────────────────────────────────────────────────────┘

PREGUNTA: ¿Ves algún proyecto listado?

SI ves un proyecto → Continúa al PASO 3
NO ves proyectos → Salta al PASO 6

═══════════════════════════════════════════════════════════════════════

🔍 PASO 3: Abrir tu proyecto
═══════════════════════════════════════════════════════════════════════

1. CLICK en tu proyecto (ej: "auditoria-db" o como lo hayas llamado)
2. Espera que cargue el dashboard del proyecto

═══════════════════════════════════════════════════════════════════════

📊 PASO 4: Ir a Table Editor
═══════════════════════════════════════════════════════════════════════

En el menú lateral izquierdo verás:

┌────────────────────────┐
│ 📊 Dashboard          │
│ 🔑 Authentication     │
│ 📁 Database           │
│    ├ Tables           │  ← CLICK AQUÍ
│    └ Backups          │
│ 🎨 Storage            │
│ ⚙️  Settings          │
└────────────────────────┘

1. CLICK en "Database" → "Tables" (o directamente "Table Editor")

═══════════════════════════════════════════════════════════════════════

🎯 PASO 5: Buscar la tabla "respuestas"
═══════════════════════════════════════════════════════════════════════

En Table Editor verás una lista de tablas:

OPCIÓN A - SI VES TABLAS:
──────────────────────────────────────────────────────────────────────

┌─────────────────────────────────────────────────────────┐
│  Tables                                    [+ New]      │
├─────────────────────────────────────────────────────────┤
│  📋 respuestas                    (150 rows)            │
│  🛒 punto_compra                  (45 rows)             │
│  👤 auth.users                    (0 rows)              │
└─────────────────────────────────────────────────────────┘

✅ SI VES "respuestas" o "punto_compra" → ¡USAS SUPABASE!

1. CLICK en "respuestas"
2. Verás datos como:
   - id, fecha, formato, local, usuario, q4_guardia_saludo, etc.
3. ¿Ves el registro "TEST-CODEPUPPY" que creé hace rato?

RESPUESTA: SÍ VEO TABLAS CON DATOS

──────────────────────────────────────────────────────────────────────

OPCIÓN B - SI NO VES TABLAS (lista vacía):

┌─────────────────────────────────────────────────────────┐

├─────────────────────────────────────────────────────────┤
│                                                         │
│      No tables found                                    │

│                                                         │
└─────────────────────────────────────────────────────────┘

❌ NO USAS SUPABASE (BD está vacía)

RESPUESTA: NO VEO TABLAS
